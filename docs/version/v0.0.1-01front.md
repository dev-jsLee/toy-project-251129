## 발생한 오류 및 해결

### 오류 상황
**문제**: "할 일 추가" 버튼에 마우스를 올리면 좌측에서부터 가림막이 글자를 가리는 애니메이션이 동작함

**원인**: 
- 버튼 호버 효과를 구현하기 위해 `::before` pseudo-element를 사용하여 그라데이션 오버레이를 추가했음
- `::before` 요소가 `z-index: 0`으로 설정되어 있었지만, 버튼 텍스트가 직접적인 자식 요소가 아니어서 `z-index: 1`이 제대로 적용되지 않음
- 결과적으로 `::before` 요소가 텍스트 위에 렌더링되어 글자가 가려짐

### 해결 방법
**수정 내용** (`frontend/css/components/button.css`):
```css
/* 수정 전 */
.button--primary::before {
    content: '';
    position: absolute;
    /* ... 그라데이션 오버레이 애니메이션 ... */
    z-index: 0;
}
.button--primary > * {
    position: relative;
    z-index: 1;
}

/* 수정 후 */
.button--primary:hover {
    background: var(--gradient-primary-light);
    box-shadow: var(--shadow-md);
    transform: translateY(-1px);
}
```

**해결 방법**:
- `::before` pseudo-element를 완전히 제거
- 호버 시 배경 그라데이션을 직접 변경하는 방식으로 단순화
- 텍스트가 항상 최상위에 렌더링되도록 보장

### 결과
- 버튼 텍스트가 정상적으로 표시됨
- 호버 효과는 유지되면서 더 간단하고 안정적인 구현
- 애니메이션 성능도 개선됨
